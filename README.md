## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `data.js`
   - `.nojekyll`
3. Go to **Settings → Pages**.
4. In **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and `/root` folder.
6. Open the GitHub Pages link after deployment.

The project is fully static: no backend and no Python runtime are required.
